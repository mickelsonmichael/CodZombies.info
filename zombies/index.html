Outlet Locations

The outlets will appear in four of the six locations listed below:

  • Aft Deck - Behind a barricaded door, to the left of a generator
  • 1st Class Lounge - Near the Zues altar
  • 3rd Class Berths - Bottom of the metal staircase on a white wall
  • Galley/Dining Room - Near the Ra altar
  • Grand Staircase - Between the clock and the symbol location
  • State Rooms - In a room beside some paintings on the floor

Outlet Order

You must use the portals created by the outlets in a particular order. The order is always as follows:

  1. Acid
  2. Water
  3. Electric
  4. Fire
